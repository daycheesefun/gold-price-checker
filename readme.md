# Readme:黄金价格监控机器人 - n8n 工作流

这是一个专门为**新手**设计的自动化工作流，帮你**零代码**搭建一个黄金价格监控系统。它会定时抓取黄金、美元指数和原油价格，当满足你设定的买入条件时，自动发送邮件提醒。

💡 本 README 记录了搭建过程中的所有坑和解决方法，跟着做一定能跑通！

## 📦 项目功能

⏰ **定时运行**：北京时间周一至周五 9:00-17:00 每小时自动执行

📈 **数据采集**：从 Yahoo Finance 免费获取黄金、美元指数、布伦特原油价格

🧠 **智能判断**：根据你设定的规则判断是否到了你设定的黄金买入点

📧 **邮件提醒**：条件满足时发邮件通知你

## 🚀 快速开始（三步搞定）

### 第一步：获取工作流文件

点击本仓库中的 gold signal workflow.json 文件，再点击右上角的 **"Download raw file"** 下载到电脑。

### 第二步：导入 n8n

1.登录你的 n8n（云端或自托管）。

2.点击右上角 **"Import from file"**，选择刚才下载的 JSON 文件。

3.导入后你会看到所有节点已经排列好，像这样：

Schedule Trigger → 获取黄金价格 → 获取美元指数 → 获取原油价格 → Set → IF → Send an email

### 第三步：配置邮箱

#### 如果你用 QQ 邮箱

登录 [QQ邮箱网页版](https://mail.qq.com/)。

点击顶部 **"设置"** → **"账户与安全"**→ **"安全设置"**。

向下找到 **"POP3/IMAP/SMTP/Exchange/CardDAV 服务"**，点击 **"开启"**。

按提示用手机发送短信，获取 **16位授权码**（**不是你的登录密码**），复制保存。回到 n8n，点击 **Send an email** 节点，在右侧 **"Credential to connect with"** 下拉框选择 **"- Create New -"**，填写：

| 字段  | 内容  |
| --- | --- |
| Name | 随便写（如 QQ邮箱） |
| Host | smtp.qq.com |
| Port | 465 |
| Secure | **勾选** |
| User | 你的完整 QQ 邮箱地址 |
| Password | 刚才复制的 **16位授权码** |

继续填写邮件内容：

**From Email**：你的 QQ 邮箱

**To Email**：接收提醒的邮箱（可以是同一个）

**Subject**：黄金买入信号触发

**Body**：用下面的模板（点击右侧齿轮 → Add Expression 插入数据）：

买入条件已满足！

当前市场数据：

- 黄金价格：{{ $node['Set'].json.gold_price }} 美元/盎司

- 美元指数：{{ $node['Set'].json.dxy }}

- 原油价格：{{ $node['Set'].json.oil_price }} 美元/桶

- 时间：{{ $node['Set'].json.timestamp }}

请登录交易软件查看。

⚠️ 注意：$node['Set'] 中的 Set 必须和你画布上合并数据的节点名称一字不差（默认就是 Set，如果你改过请相应修改）。点击节点右上角的 "Execute Step" 测试，如果能收到邮件，配置成功！

## 🔧 详细配置说明（小白必看）

### 1. 定时触发器（Schedule Trigger）

时区已设为 **Asia/Shanghai**（北京时间）。

Cron 表达式：0 9-17 * * 1-5 → 周一到周五 9点到17点。

如果你想改时间，双击该节点，在 **"Expression"** 里修改即可。

### 2. 数据源节点（HTTP Request）

三个节点分别获取：

黄金：GC=F

美元指数：DX-Y.NYB

原油：BZ=F

它们都使用 Yahoo Finance 的公开 API，**不需要任何 API Key**，完全免费。

### 3. 合并数据节点（Set）

这个节点把三个数据合并成一个对象，方便后面使用。它的输出是：

json

{ "gold_price": 5180.6, "dxy": 98.958, "oil_price": 83.69, "timestamp": "2026-03-05T12:00:00Z"}

### 4. 判断节点（IF）

目前设置的买入规则是：

黄金价格 **≤ 5000**

美元指数 **< 100**

原油价格 **> 70**

你可以随时双击 IF 节点修改这些阈值。

### 5. 邮件节点（Email）

邮件正文中的表达式必须用 **双花括号** {{ ... }}

## ❓ 常见问题与解决方法（全是过来人的经验）

### Q1：注册 GoldAPI / TwelveData 时死活过不去人机验证？

**解决**：直接改用 Yahoo Finance，不需要注册。本工作流已经内置了 Yahoo 的接口，你不需要再注册任何数据服务。

### Q3：邮件发不出去，报错 535 Login fail？

**解决**：99% 是因为你没用授权码，或者授权码过期。重新生成一个新授权码，在 n8n 凭证里更新。另外检查端口必须是 465，且勾选 SSL/TLS。

### Q6：工作流右上角找不到"Active"开关？

**解决**：你可能在分享视图里。点击左上角 **"Overview"** 回到工作流列表，再点击工作流名称进入编辑模式。点击上面的**Publish**就是Active的功能，点击激活后程序可以按照schedule trigger的时间设定自动运行。

### Q7：Yahoo Finance 的数据会不会突然不能用？

**解决**：Yahoo 是成熟的金融平台，非常稳定。万一接口有变动，你可以随时修改 HTTP 节点的 URL，换成其他免费源（如 Alpha Vantage），但需要自己申请 API Key。

## ✏️ 自定义你的规则

想要修改买入条件？很简单：

1.双击 **IF** 节点。

2.在 **"Conditions"** 区域修改数值或操作符（如把黄金价格区间改成 4800-5100）。

3.点击空白处保存，然后点击 **"Save"** 保存整个工作流。

你也可以增加条件，比如要求"美元指数连续两天下跌"，但那就需要额外节点了。目前四个条件已经够用。
